# csharp

這是 C# 版本的 LeetCode 解題專案。

- 解答放在 `LeetCode/Solutions/`
- 測試放在 `LeetCode.Test/`
- 題目檔名用 `S####.cs`，例如 `S0001.cs`
- 測試檔名用 `S####Tests.cs`，例如 `S0001Tests.cs`
- 共用的 List/Tree/Graph 工具放在 `LeetCode/Utilities/`

AI 注意：改某題時只動對應的 `S####.cs` 和 `S####Tests.cs`，不要碰無關題目。遇到 ListNode、TreeNode、Graph 題，先看 `Utilities/` 是否已有工具。
